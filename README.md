# Customer_Churn_Prediction_Model

# Customer Churn Prediction using Machine Learning

## Project Overview

This project focuses on predicting customer churn using the Telco Customer Churn dataset. The main goal is to identify customers who are likely to discontinue telecom services and analyze the key factors influencing churn.

## Business Problem

Customer churn directly affects company revenue and customer lifetime value. By predicting churn in advance, businesses can take proactive retention actions such as offering discounts, improving service quality, and targeting high-risk customers with personalized strategies.

## Dataset

The dataset contains customer information such as:

* Customer demographics
* Tenure
* Contract type
* Internet service
* Monthly charges
* Total charges
* Payment method
* Churn status

## Tools and Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Scikit-learn
* Jupyter Notebook

## Project Workflow

1. Loaded and understood the dataset
2. Handled missing values and data type issues
3. Performed exploratory data analysis
4. Visualized churn patterns across different customer groups
5. Encoded categorical variables
6. Built machine learning models for churn prediction
7. Evaluated model performance using accuracy, precision, recall, F1-score, and confusion matrix

## Key Insights

* Customers with month-to-month contracts showed higher churn rates.
* Customers with shorter tenure were more likely to churn.
* Higher monthly charges were associated with increased churn probability.
* Electronic check payment users showed higher churn compared to other payment methods.

## Machine Learning Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* XGBoost / Gradient Boosting

## Results

The model successfully classified customers into churn and non-churn categories. The analysis helped identify high-risk customer segments and provided actionable insights for customer retention.

## Business Impact

This project demonstrates how machine learning can help telecom companies reduce customer churn, improve customer retention, and increase revenue by targeting at-risk customers before they leave.
